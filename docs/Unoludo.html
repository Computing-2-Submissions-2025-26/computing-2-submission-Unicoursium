Unoludo

Namespace

Unoludo

Description:

  • A pure JavaScript game module for Unoludo, a turn-based board game combining UNO-inspired card play with Ludo-inspired plane movement.

    The API creates immutable game states and returns a new state whenever a legal card play, draw, movement, capture, or turn transition is made. Illegal moves return undefined so the user interface can reject them without mutating the existing game.

(static) can_play_on(card, top_card, active_colour) → {boolean}

Description:

  • Return whether a card can be played on the current discard state.

    A card can be played if:

    • it is a Wild or Wild +4 card;
    • it matches the active colour;
    • it is a number card with the same value as the top number card;
    • it is an action card with the same action type as the top action card.
